# sa_casino

South African Casino is a React/Vite prototype for the card game setup screen.

## Run locally

```bash
npm install
npm run dev
```

## Included

- 40-card deck with Ace through 10 in four suits
- 2, 3, or 4 player deal setup
- Random first dealer
- Human seat at the bottom, CPU seats around the table
- Face-up table card for 3-player deals
- Player state scaffolding for hand, capture pile, and builds
